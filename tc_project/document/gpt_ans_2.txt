# 十九、构造序列数据

---

# 代码

```python
def create_sequence(data, target, seq_len=24):

    X = []
    y = []

    for i in range(len(data)-seq_len):

        X.append(data[i:i+seq_len])

        y.append(target[i+seq_len])

    return np.array(X), np.array(y)
```
