# C题《量化投资问题》完整解决方案

## 一、问题背景与任务目标

本题要求基于 2021 年 7 月 14 日至 2022 年 1 月 28 日的“数字经济”板块 5 分钟级别高频数据，结合宏观经济指标、国内外市场指标、技术指标及相关板块信息，构建量化投资模型。

核心目标包括：

1. 提取与“数字经济”板块高度相关的关键指标；
2. 预测数字经济板块 5 分钟成交量；
3. 预测数字经济板块 5 分钟收盘价；
4. 基于预测结果构建量化交易策略并评价收益表现。

---

# 二、整体技术路线

整体方案采用：

# “特征筛选 + 多模型融合预测 + 高频量化交易”

总体流程如下：

```text
数据读取
   ↓
数据清洗与时间对齐
   ↓
特征工程
   ↓
相关性分析与指标筛选
   ↓
成交量预测模型
   ↓
收盘价预测模型
   ↓
交易信号生成
   ↓
回测系统
   ↓
收益与风险评价
```

因此必须进行：

# 时间尺度统一与缺失值处理

# 四、数据预处理

## 4.1 时间统一

统一时间索引：

```python
DatetimeIndex
```

将所有数据按 5 分钟频率对齐。

对于低频宏观指标：

采用：

# 前向填充（Forward Fill）

即：

```text
某月 PMI → 填充到该月所有 5 分钟数据
```

原因：

宏观经济指标在短时间内变化缓慢。

---

## 4.2 缺失值处理

### （1）连续型数据

采用：

* 前向填充
* 后向填充
* 插值法

### （2）异常值处理

采用：

# 3σ原则

定义：

```math
|x - μ| > 3σ
```

则视为异常值。

处理方法：

* Winsorize 截尾
* 中位数替换

---

## 4.3 数据标准化

由于不同指标量纲差异巨大：

例如：

* RSI：0~100
* 成交额：上亿
* 汇率：个位数

因此采用：

# Z-score 标准化

公式：

```math
x'=(x-μ)/σ
```

问题一的核心：

# 从大量指标中筛选出对数字经济板块影响最大的特征

这是典型：

# 高维特征筛选问题

# 六、特征工程

## 6.1 构造收益率特征

定义：

```math
Return_t = (P_t - P_{t-1}) / P_{t-1}
```

构造：

* 5分钟收益率
* 30分钟收益率
* 日内累计收益率

---

## 6.2 波动率特征

定义：

```math
Volatility_t = Std(Return)
```

构造：

* 滚动波动率
* ATR
* 布林带宽度

---

## 6.3 技术指标增强

重点保留：

| 指标   | 含义   |
| ---- | ---- |
| MACD | 趋势   |
| RSI  | 超买超卖 |
| KDJ  | 动量   |
| BOLL | 波动   |
| OBV  | 量价关系 |
| MA   | 趋势均值 |

---

## 6.4 时间周期特征

构造：

* hour
* minute
* weekday
* 是否午盘
* 是否开盘前30分钟

原因：

高频市场具有明显时间规律。

# 七、相关性分析

## 7.1 Pearson相关系数

计算：

```math
ρ = Cov(X,Y)/(σ_Xσ_Y)
```

筛选：

```text
|ρ| > 0.3
```

的特征。

## 7.2 灰色关联分析

由于金融数据非线性较强。

因此进一步采用：

# 灰色关联分析（GRA）

计算各指标与目标序列的关联度。

关联度越高：

说明指标影响越大。

---

## 7.3 随机森林特征重要性

建立：

```text
Random Forest Regressor
```

利用：

```python
feature_importances_
```

排序。

最终得到 TOP 特征。

# 九、问题二：成交量预测模型

目标：

预测：

# 数字经济板块未来5分钟成交量

---

# 十、成交量预测建模思路

成交量具有：

* 强时间序列性
* 明显周期性
* 自相关性强
* 高频噪声大

因此采用：

# “统计模型 + 深度学习模型”融合

---

# 十一、模型一：XGBoost

XGBoost适合：

* 非线性关系
* 多因子建模
* 高维特征

输入：

```text
历史成交量
技术指标
指数数据
国际市场数据
```

输出：

```text
未来5分钟成交量
```

---

# 十二、模型二：LSTM

由于成交量存在时间依赖。

因此进一步采用：

# 长短期记忆网络（LSTM）

输入窗口：

```text
过去24个5分钟数据
```

即：

```text
过去2小时
```

输出：

```text
未来1个时间步成交量
```

LSTM优势：

* 能捕捉时间依赖
* 能处理长期记忆
* 对金融时间序列效果较好

---

# 十三、成交量模型评价指标

采用：

| 指标   | 公式        |
| ---- | --------- |
| MAE  | 平均绝对误差    |
| RMSE | 均方根误差     |
| MAPE | 平均绝对百分比误差 |
| R²   | 拟合优度      |

推荐：

# 以 RMSE 为主要指标

原因：

金融数据对大误差更敏感。

# 十五、价格预测模型设计

提升预测精度。

# 二十、交易信号设计

定义预测收益率：

```math
r_t = (P^_{t+1} - P_t)/P_t
```

其中：

* P^ 为预测价格
* P 为当前价格

---

## 20.1 买入信号

当：

```math
r_t > θ_1
```

则：

```text
买入
```

---

## 20.2 卖出信号

当：

```math
r_t < -θ_2
```

则：

```text
卖出
```

---

## 20.3 不交易

当：

```math
-θ_2 ≤ r_t ≤ θ_1
```

则：

```text
持仓不动
```

---

# 二十五、收益指标计算

## 25.1 总收益率

公式：

```math
R=(\prod(1+r_i)-1)×100%
```

---

## 25.2 信息比率

公式：

```math
IR = 平均超额收益 / 超额收益标准差
```

其中：

```math
超额收益 = 策略收益 - 0.9 × 中证500收益
```

---

## 25.3 最大回撤率

公式：

```math
MDD = (Peak - Trough)/Peak
```

反映最大亏损风险。
